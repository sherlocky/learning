* MurmurHash （来自google）
  > 是一种非加密型哈希函数，适用于一般的哈希检索操作，效率很高。  
在Guava 的 Hashing 类中有该算法实现，Redis，Memcached，Cassandra，HBase，Lucene 等都使用了它。  
可以获得一个整形结果，然后再利用62进制可实现短链接效果。
